Viper!

An object oriented python library for generating web pages.

Quickstart

Easy to Use

Viper was designed to use Python classes to build your page, which makes it really easy.

Focus on What Matters

Viper's modular style lets you add and remove modules as you need them.

Powered by Jinja2

Viper is powered by a Jinja2 backend, which makes generating source code and creating your own modules super fast and easy.
